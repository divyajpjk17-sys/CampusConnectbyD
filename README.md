# CampusConnect

A modern student community platform blending the professional depth of LinkedIn, the casual energy of Threads, and the community warmth of Discord.

## Project Structure

- `campusconnect-frontend`: React frontend using Vite, Tailwind CSS, and Framer Motion.
- `campusconnect-backend`: Node.js/Express backend using JSON files for storage.

## Getting Started

### Backend
```bash
cd campusconnect-backend
npm install
npm run dev
```

### Frontend
```bash
cd campusconnect-frontend
npm install
npm run dev
```
